Orbis Help

Importing and Exporting Data

Exporting your results

Orbis allows you to export the results of your calculations to Comma Seperated Value (.csv) format that can be read by all spreadsheet programs. To export the Huckel determinant and all of the numerical results of your calculation, simply choose the "Export Results..." (or use Ctrl+E) option from the File menu.

Importing data

Importing molecular geometry data: Orbis can currently import molecular geometry information from the following chemical file formats:

  • MDL Molfiles (.mol)
  • XYZ files
  • tab,space or comma delimited text files of atomic types and coordinates (e.g each line of the file would contain a record like C,1.0,1.0,0.0)
Note that the Z coordinate is always assumed to be 0. For XYZ and generic text files Orbis tries to guess where bonds are located and these may need to be adjusted manually after import.

Importing a Huckel determinant:Orbis also allows you to import a Huckel determinant from a text file and will automatically construct a molecule for you based on the determinant. For example if you had a text file with the following format:
